**Mathematical Justification:**
Let the security of a system be `S`. A single-layer defense is a function `f(x)` that returns `true` if input `x` is valid. The system is secure if `f(x_attack)` is `false`. A multi-layered defense is a composite function `S = f_n(...f_2(f_1(x))...)`.

**Proof of Resilience:** The probability of failure for a single layer is `P(f_i fails) = p_i`. For a single-layer system, the total probability of failure is `p_1`. For the multi-layered system, the total probability of failure is approximately `Π p_i` (assuming independence). Since each `p_i < 1`, the product `Π p_i` is significantly smaller than any individual `p_i`. A layered defense is thus proven to be exponentially more resilient than a single perimeter defense. `Q.E.D.`
