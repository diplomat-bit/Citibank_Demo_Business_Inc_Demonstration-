## Architectural Supremacy

This platform was not merely built; it was engineered for dominance.
- **Framework**: **React 19**, leveraging its latest concurrency and performance features.
- **AI Core**: **Google Gemini API**, utilizing the full power of `gemini-2.5-flash`, `imagen-4.0-generate-001`, and `veo-2.0-generate-001` for text, image, and video.
- **State Management**: A monolithic, "God-file" **React Context (`DataContext.tsx`)** serves as the absolute source of truth, a design pattern chosen for its raw power and centralized control in a complex system of this scale.
- **Architecture**: A fully modular, enterprise-grade component architecture. The entire system is built upon a robust custom component library and a scalable, domain-driven type system. The `openapi.yaml` and `SPEC_CODEX.md` in this repository further detail this professional-grade foundation.
- **Visuals**: Styled with **Tailwind CSS** for rapid, utility-first development and visualized with **Recharts** for dynamic, data-rich charting.
