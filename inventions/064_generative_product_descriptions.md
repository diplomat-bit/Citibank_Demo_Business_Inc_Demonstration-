
**Title of Invention:** System and Method for Generating E-commerce Product Descriptions

**Abstract:**
A system for creating e-commerce content is disclosed. A user provides a list of key features or bullet points for a product. The system sends these points to a generative AI model, prompted to act as a marketing copywriter. The AI expands the bullet points into a full, compelling, and SEO-friendly product description.

**Detailed Description:**
In the Commerce module, a user enters bullet points for a new product. They click "AI Write Description." The system prompts an LLM: `You are a copywriter. Write an exciting product description from these features: [features].` The AI's text is then populated into the product description field.

**Claims:**
1. A method for creating product content, comprising:
   a. Receiving a list of product features.
   b. Transmitting the features to a generative AI model.
   c. Prompting the model to generate a full-length product description.
   d. Displaying the description to a user.
